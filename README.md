# 🏮 量子动效手势控制网站

一个具有春节中国风设计风格的量子动效网站，支持手势和鼠标交互控制。

## ✨ 特性

- 🎯 **手势控制**: 使用摄像头捕捉手部动作，控制量子动效位置
- 🖱️ **鼠标控制**: 支持鼠标移动控制，随时切换模式
- 🏮 **春节中国风**: 采用春节主题配色（红色、金色）和中国传统元素
- ✨ **量子动效**: 流畅的粒子效果、光晕动画和量子波动
- 📱 **响应式设计**: 适配桌面和移动设备
- 🎨 **Tailwind CSS**: 使用 Tailwind 进行样式设计

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看效果。

### 构建生产版本

```bash
npm run build
npm start
```

## 📦 技术栈

- **Next.js 15** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **MediaPipe Hands** - 手势识别
- **Canvas API** - 粒子动效

## 🎮 使用说明

1. **首次访问**: 页面会请求摄像头权限，请允许访问以启用手势控制
2. **手势模式**: 在摄像头画面中移动手部，量子动效会跟随手心位置
3. **鼠标模式**: 点击页面切换回鼠标控制模式
4. **视觉反馈**: 右下角显示摄像头画面和手部骨架追踪

## 🎨 设计主题

- **主色调**: 红色 (#E60000) - 春节传统色
- **辅助色**: 金色 (#FFD700) - 财富与吉祥
- **装饰元素**: 灯笼、红包、烟花等春节元素
